## 3.1 简述JDK,JRE,JVM之间的关系

   * JDK(Java Development Kit) 是 Java 语言的软件开发工具包
   * Jre是Java运行环境
   * JVM是一种用于计算设备的规范,是通过在实际的计算机上仿真模拟各种计算机功能来实现的

## 3.2 描述类加载机制的含义和过程

   * 类加载：通过类的全限定名获取其定义的二进制字节流，
   * 需要借助类加载器完成，就是用来装载class文件的，通过双亲委派机制加载
